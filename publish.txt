
Howto distribute via PyPi
--------------------------------------

in terminal -> cd C:\Users\Bert\PycharmProjects\rpio

- py -m pip install build
- py -m build --sdist
- twine check dist/*
- twine upload dist/<specific-distribution>   -> username = __token__ password = see local file

Installing the distribution
---------------------------
- pip install Interface-KB


See: https://packaging.python.org/en/latest/guides/distributing-packages-using-setuptools/#pure-python-wheel
